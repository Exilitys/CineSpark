## ðŸ§ª Local Development

### 1. Clone the repository:
```bash
git clone https://github.com/Exilitys/CineSparkAI.git
cd CineSparkAI
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Set up environment variables:

Create a `.env` file in the root directory:

